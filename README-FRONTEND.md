Kortfattad beskrivning

Tagga dig är en tjänst som skall underlätta för företag genom att erbjuda enkel bokning utav reklamskyltar i göteborg.
Skyltarna kan vara antingen statiska eller rörliga och visas upp på olika typer utav monitorer.

Teknologier vi använt oss utav:

	* Bootstrap
	* AJAX
	* JQuery
	* Google maps api

Arbetsprocess

Vi började med att utveckla sidan för desktop, därefter anpassade vi den för mobila enheter. 
Det blev så eftersom det var en naturlig start då vi hade fått en mockup som var i desktop format.
Parprogrammering var någonting som var nytt för oss alla men som vi tyckte fungerade väldigt bra, 
det gjorde att vi sällan satt fast på grund av slarvfel så som dålig syntax eller liknande.
Vi har bollat ideer en hel del och på så vis kommit fram till en hel del bra lösningar.
Sammarbetet har gått väldigt bra, någonting som vi kan bli bättre på är versionshanteringen samt ett ännu tydligare upplägg på vem som skall göra vad.
